## 🚀 Быстрый старт

### Android — v2rayNG

1. Установи [Happ](https://github.com/Happ-proxy/happ-android) из GitHub
2. Открой приложение → нажми **+** в правом верхнем углу
3. Выбери **Импорт конфигурации из URL**
4. Вставь ссылку подписки → **OK**
5. Нажми ▶️ для подключения

### iOS — Streisand / Shadowrocket

1. Установи [Streisand](https://apps.apple.com/app/streisand/id6450534064) (бесплатно) или Shadowrocket
2. Открой приложение → **+** → **Тип: Subscribe**
3. Вставь ссылку подписки → **Сохранить**
4. Нажми **Обновить** и выбери сервер

### Windows / macOS — Hiddify

1. Скачай [Hiddify](https://github.com/hiddify/hiddify-app/releases)
2. Нажми **+** → **Добавить из буфера / URL**
3. Вставь ссылку подписки → **Добавить**
4. Выбери любой сервер и подключись
